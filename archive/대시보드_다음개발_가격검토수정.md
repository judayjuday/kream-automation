# 대시보드 다음 개발사항 — 가격 검토/수정 기능

> 현재 큐 + 일괄 실행이 완료된 후 추가할 기능

---

## 문제

1. 일괄 실행 후 결과를 사용자가 **디테일하게 검토할 방법이 없음**
2. 자동 선택된 가격/마진을 사용자가 **수정할 수 없음**
3. 메인 상품이나 중요한 상품은 사이즈별/가격별로 **상세 분석이 필요**

---

## 구현할 기능

### 1. 상세 가격 분석 버튼

큐 결과 테이블에서 각 상품 옆에 **"📊 상세"** 버튼 추가.

### 2. 가격 직접 수정

결과 테이블에서:
- **입찰가 컬럼을 클릭하면 직접 금액 입력 가능** (인라인 편집)
- 또는 📝 모달에서 입찰가를 직접 입력 가능
- 수정하면 마진이 실시간 재계산되어 표시
- 자동 선택된 마진율(10%/15%/20%) 말고 "직접 입력" 옵션 추가

### 3. 신발 사이즈별 가격 개별 수정

신발 상품의 펼치기 뷰에서:
- 각 사이즈의 입찰가를 개별 수정 가능
- 사이즈별로 다른 마진율 적용 가능
- 특정 사이즈만 체크/체크해제로 입찰 대상에서 제외

### 4. 메인 상품 표시

- 큐에 추가할 때 "⭐ 메인 상품" 체크 옵션
- 메인 상품은 결과 테이블에서 강조 표시
- 메인 상품은 일괄 실행 후 자동으로 상세 분석 패널이 열림
- 메인 상품이 아닌 건 기본 마진율로 자동 처리

---

## 즉시 할 작업

1. 결과 테이블에 "📊 상세" 버튼 추가 → 가격 상세 분석 패널
2. 입찰가 인라인 편집 (클릭하면 수정 가능, 수정 시 마진 실시간 재계산)
3. 📝 모달에 "직접 입력" 가격 옵션 추가
4. 신발 사이즈별 입찰가 개별 수정
5. 큐 추가 시 "⭐ 메인 상품" 체크 옵션
