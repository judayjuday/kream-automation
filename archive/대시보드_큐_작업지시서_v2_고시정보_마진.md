## 1. 고시정보 자동 입력

### 현재 고시정보 필드 (KREAM 판매자센터 기준)

상품마다 다른 것만 입력:
- **종류** — 숄더백, 토트백, 크로스백... (가방) / 러닝화, 스니커즈... (신발)
- **소재** — 데님, 가죽, 나일론... / 합성피혁, 메쉬...
- **색상** — 블루, 블랙...
- **크기** — 50*20*30cm 또는 상품별 상이 / 260mm 등
- **제조자/수입자** — Adidas, Nike...

나머지는 설정 기본값 자동 적용 (이미 구현됨):
- 제조국: "상품별 상이 (케어라벨 참고)"
- 주의사항: 기본 문구
- 품질보증: "관련 법 및 소비자 분쟁 해결 기준에 따름"
- AS전화번호: "01075446127"
- 원산지: "China (중국) (CN)"
- HS코드: 가방 "4202.92" / 신발 "6404.11"

### 자동 채움 로직

KREAM 검색 결과에서 가져올 수 있는 것:
```python
def auto_fill_product_info(kream_data, category):
    """KREAM 검색 결과 + 카테고리로 고시정보 자동 채움"""
    
    info = {}
    
    # 제조자/수입자 — KREAM 상품명 또는 브랜드에서 추출
    info["manufacturer"] = kream_data.get("brand", "")
    # 예: "adidas", "Nike", "Asics", "New Balance"
    
    # 색상 — KREAM 옵션명에서 추출
    # KREAM 옵션명 예: "Core Black/Cloud White" → "블랙, 화이트"
    info["color"] = extract_color(kream_data.get("option_name", ""))
    
    # 카테고리별 기본값
    if category == "가방":
        info["type"] = "상품별 상이"  # 종류
        info["material"] = "상품별 상이"  # 소재
        info["size"] = "상품별 상이"  # 크기
        info["hs_code"] = "4202.92"
        info["tariff"] = 8
    elif category == "신발":
        info["type"] = "운동화"
        info["material"] = "합성피혁, 고무"
        info["size"] = "상품별 상이"
        info["hs_code"] = "6404.11"
        info["tariff"] = 13
    elif category == "의류":
        info["type"] = "상품별 상이"
        info["material"] = "상품별 상이"
        info["size"] = "상품별 상이"
        info["hs_code"] = ""  # 의류는 다양
        info["tariff"] = 13
    
    # 고시정보 등록 여부 체크
    info["goshi_registered"] = False  # KREAM에서 확인 가능하면 업데이트
    
    return info
```

### 고시정보 등록 여부 확인

- KREAM 판매자센터에서 해당 상품의 고시정보가 이미 등록되어 있는지 확인
- 이미 등록됨 → "이미 고시정보 등록됨 (입찰만 진행)" 체크박스 자동 체크
- 미등록 → 자동 채운 고시정보로 등록 진행

## 2. 일괄 실행 결과 테이블 (마진 확인 + 가격 선택 통합)

일괄 실행 후 하나의 큰 테이블에서 모든 것을 한눈에 처리:

### 테이블 구성

```
┌────┬──────────┬─────────────┬──────┬────────┬──────────┬─────────┬─────────┬─────────┬──────┬────────┬────┐
│ ☑  │ 모델번호 │   상품명    │카테고│원가(원)│KREAM최저가│마진10% │마진15% │마진20% │고시정보│ 상태  │액션│
├────┼──────────┼─────────────┼──────┼────────┼──────────┼─────────┼─────────┼─────────┼──────┼────────┼────┤
│ ☑  │ IX7694   │ Adidas Bag..│ 가방 │ 63,000 │  89,000  │+12,000 │ +7,500 │ +3,000 │ 미등록│  완료  │ 📝 │
│ ☐  │ ID6016   │ VL Court... │ 신발 │ 85,000 │  79,000  │ -8,500 │-13,500 │-18,500 │ 등록됨│  완료  │ 📝 │
│ ☑  │ 1203A408 │ Asics Gel...│ 신발 │ 78,000 │ 120,000  │+28,000 │+22,000 │+16,000 │ 미등록│  완료  │ 📝 │
└────┴──────────┴─────────────┴──────┴────────┴──────────┴─────────┴─────────┴─────────┴──────┴────────┴────┘
```

### 각 컬럼 설명

- **☑ (체크박스):** 등록/입찰할 상품 선택. 마진이 마이너스인 건 자동으로 체크 해제
- **모델번호:** 입력한 품번
- **상품명:** KREAM 검색으로 가져온 이름 (짧게 표시, 호버하면 풀네임)
- **카테고리:** 가방/신발/의류 — 클릭하면 드롭다운으로 변경 가능
- **원가(원):** CNY → KRW 환산 + 관세 + 배송비 합산
- **KREAM 최저가:** 현재 KREAM의 최저 판매 입찰가
- **마진 10%/15%/20%:** 각 마진율에서의 예상 수익(원). 양수면 초록, 음수면 빨강
- **고시정보:** "등록됨" / "미등록" 표시. 미등록이면 자동 채움된 정보로 등록 예정
- **상태:** 대기/검색 중/계산 중/완료/실패
- **액션 (📝):** 클릭하면 해당 상품의 상세 편집 모달 열림

### 상세 편집 모달 (📝 클릭 시)

각 상품을 개별 수정할 수 있는 모달:

```
┌─────────────────────────────────────────┐
│ IX7694 — Adidas Bag...           [닫기] │
├─────────────────────────────────────────┤
│                                         │
│ 기본 정보                               │
│ ├ 카테고리: [가방 ▼]                    │
│ ├ 중국 구매가: [220] CNY                │
│ ├ 해외배송비: [8000] 원                 │
│ ├ 사이즈: [ONE SIZE]                    │
│ └ 수량: [1]                             │
│                                         │
│ 고시정보                                │
│ ├ 종류: [상품별 상이]                   │
│ ├ 소재: [상품별 상이]                   │
│ ├ 색상: [블랙]       ← 자동 채움       │
│ ├ 크기: [상품별 상이]                   │
│ └ 제조자: [Adidas]   ← 자동 채움       │
│                                         │
│ 마진 상세                               │
│ ├ 원가: 63,000원                        │
│ │  (CNY 220 × 218.12 × 1.03           │
│ │   + 관세 8% + 배송 8,000)            │
│ ├ KREAM 최저가: 89,000원               │
│ ├ 손익분기 판매가: 70,000원            │
│ ├ 마진10% 판매가: 77,000원 (+7,000)    │
│ ├ 마진15% 판매가: 80,500원 (+10,500)   │
│ └ 마진20% 판매가: 84,000원 (+14,000)   │
│                                         │
│ 입찰 가격 선택                          │
│ ○ 손익분기  ○ 마진10%  ● 마진15%      │
│   → 입찰가: 80,500원                   │
│   → 예상 수익: +10,500원               │
│                                         │
│         [저장] [취소]                   │
└─────────────────────────────────────────┘
```

### 가격 선택 방식

두 가지 모드:

**모드 A — 일괄 기본 마진 설정 (빠른 처리)**
- 테이블 상단에 "기본 마진율: [15%▼]" 드롭다운
- 선택하면 전체 상품에 해당 마진율 적용
- 개별 수정 필요하면 📝 모달에서 변경

**모드 B — 개별 가격 선택 (정밀 처리)**
- 📝 모달에서 상품별로 마진율 또는 직접 가격 입력

기본은 모드 A로 빠르게 처리하고, 필요한 것만 모드 B로 세밀하게 조정.

## 3. 등록/입찰 실행

**"선택 상품 등록/입찰"** 버튼 클릭 시:

```
각 선택된 상품마다:
1. 고시정보 미등록 → 고시정보 자동 등록 (kream_bot.py의 product 모드)
2. 고시정보 등록됨 → 건너뜀
3. 선택된 가격으로 판매 입찰 등록 (kream_bot.py의 bid 모드)
4. 상태 업데이트: 등록 중 → 입찰 중 → 완료/실패
```

이 단계는 Playwright로 KREAM 판매자센터를 실제 조작하는 것이므로, 
대시보드에서 "실행" 버튼을 누르면 백엔드가 kream_bot.py를 호출한다.
