# 🆕 신규 입찰 도구 사용 가이드 (Step 31에서 만든 거 활용)

## 1. 진입

브라우저에서 http://localhost:5001 → 사이드바 "🆕 신규 입찰" 메뉴 클릭

## 2. 모델 입력

textarea에 **한 줄에 모델 1개 + CNY 가격** 형식으로 입력:

```
JQ4110	350
1183B938-100	626
IX7693	280
```

- 모델번호와 CNY 사이는 **TAB 키** (스페이스 X)
- 엑셀에서 복사 붙여넣기 가능 (엑셀 셀 사이는 자동으로 TAB)
- 한 번에 최대 20개

## 3. "시장가 수집" 버튼

KREAM에서 각 모델 시장가 자동 수집 (모델당 약 30초, 한국에서 가능)

## 4. 마진 결과 확인

테이블이 자동으로 채워짐:

| 모델 | 사이즈 | 시장가 | CNY | 입찰가 | 원가 | 마진 | 판정 |
|---|---|---|---|---|---|---|---|
| JQ4110 | 220 | 130,000 | 350 | 129,000 | 84,067 | **38,189원** | ✅ GO |
| JQ4110 | 215 | 95,000 | 350 | 94,000 | 84,067 | 4,733원 | ⚠️ LOW |
| JQ4110 | 270 | 60,000 | 350 | 59,000 | 84,067 | -33,627원 | ❌ DEFICIT |

- **GO** (녹색): 마진 4,000원 이상, 자동 체크됨
- **LOW** (노란색): 0~4,000원, 단가 협상 필요
- **DEFICIT** (빨간색): 적자, 입찰 불가

## 5. "선택 항목 큐 등록" 버튼

체크된 항목만 큐에 등록.
- 기본은 GO만 자동 체크
- 사장이 수동으로 LOW도 체크 가능 (전략적 판단)

## 6. 큐에서 자동 입찰

사이드바 "📦 상품 등록/입찰" 메뉴 → 큐 목록 확인 → "자동 입찰 실행" 버튼

(큐 등록 시 bid_cost도 자동 저장됨 → 나중에 판매되면 마진 확정 데이터에 자동 매칭)

## 7. 매출 확인

사이드바 "💰 판매 마진 확정" 메뉴:
- 월별 매출 막대 차트
- 모델별 ROI 순위
- 현재 누적: 매출 689,000원 / 확정 마진 56,746원 / 마진율 8.2%

## 8. 시장 변화 감지

사이드바 "📈 시장 모니터링" 메뉴:
- 활성 모델 가격 추이 (2시간마다 자동 수집)
- 가격 ±10% 이상 급변 알림
- 24시간 후 의미 있는 데이터 누적됨

## 자주 묻는 질문

**Q: GO인데 왜 마진이 낮게 나오나?**
A: 시장 즉시구매가 - 1,000원 언더컷이 입찰가. KREAM 수수료 6% + 부가세 0.6% + 고정 2,500원 + 원가 (CNY×환율×1.03 + 8,000 배송비) 빼면 마진. 사이즈별로 시장가 차이 큼.

**Q: 같은 모델인데 사이즈별 GO/LOW 다른 이유?**
A: 사이즈마다 즉시구매가 다름. 인기 사이즈는 시장가 높아서 마진 좋고, 비인기 사이즈는 낮아서 LOW.

**Q: CNY 어디서 확인?**
A: Dewu(得物) 앱에서 모델번호 검색 → 최저가 (배송비 별도 8,000원 포함됨).

**Q: LOW 항목 입찰해도 되나?**
A: 케이스 바이 케이스. 회전 빠른 인기 모델이면 OK (월 5건 팔리면 4,000×5 = 20,000원 누적). 회전 느린 모델은 비추.

## 다음 단계 (사장이 직접)

1. 신규 모델 1개 골라서 위 워크플로우 진행
2. 큐 등록까지 해보고 자동 입찰 실행
3. 결과 (몇 건 입찰됐는지) 확인
4. 1주일 후 판매 마진 확정 탭에서 확인
