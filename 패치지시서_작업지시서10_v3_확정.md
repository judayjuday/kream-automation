### 수정 2: Step 5 UI 단일화 — 미니멈 인라인

- 또는 tab_settings.html 하단 "식货 도구" 섹션
- 권장: tab_adjust.html 상단에 작은 섹션 추가 ("식货 활성 batch / 차이 보기")

검증 명령:
```bash

```
실측 후 정확한 삽입 위치(파일+라인)를 v3 본문에 명시.

UI 사양 (v2와 동일, 위치만 확정):
- 버튼: "식货 ↔ 등록 원가 차이 보기"

- 차이 0건이면 "차이 없음" 메시지
